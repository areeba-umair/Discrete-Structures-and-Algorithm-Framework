The CDAF System is a comprehensive C++ application that integrates core concepts of Discrete Mathematics and Graph Algorithms into a single interactive system.

It demonstrates practical implementation of:

Set Theory
Logic & Predicates
Graph Theory
Relations & Functions

Key Features
Discrete Set Implementation
Custom dynamic array (no STL set)

Supports:
Union, Intersection, Difference
Symmetric Difference
Power Set
Cartesian Product
Logic Engine

Quantifiers:
∀ (For All)
∃ (Exists)
∃! (Unique Exists)
De Morgan’s Law verification
 Graph Theory Engine
Adjacency Matrix + Adjacency List

Algorithms:
Dijkstra (Shortest Path)
Kruskal (MST)
DFS (Connected Components)
Graph Properties:
Euler Path & Circuit
Hamiltonian Cycle
Complete Graph Check
 Relations & Functions
 
Relation Properties:
Reflexive
Symmetric

Function Properties:
Injective
Bijective
Total Function
 Constraint System
 
Real-world modeling using sets:
Critical nodes
